---
title: Trusted Devices
description: Approve, use, and revoke trusted devices.
---

Trusted devices let you move encrypted account access to a new browser or device with help from a device you already trust.

## Add a device

Start the device approval flow on the new device, then approve it from an existing trusted device. The approving device creates an encrypted envelope that only the requesting device can consume.

Approvals expire. If a request is old or unexpected, deny it and start again from a device you control.

## Manage devices

Review trusted devices from account security. Revoke devices that are lost, shared, retired, or no longer yours.

Revoking a trusted device prevents future use of that device envelope. It does not erase data already copied out of a compromised browser or device, so revoke quickly when access is in doubt.
