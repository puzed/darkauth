DarkAuth is licensed under AGPL-3.0.

## What that means operationally

AGPL-3.0 is a strong copyleft license that includes network-use obligations. Teams deploying modified versions should understand the license before offering the software as a network service.

This page should eventually include a fuller license and dependency review, but the source of truth is the repository license files and package metadata.

## Package license files

Several packages include their own `LICENSE` files. Keep those aligned with the repository-level license and update this page if package licensing changes.
